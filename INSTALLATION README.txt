Welcome to Hospital Management System, this system interfaces with the provided HealthCarePro and Surgery Clinic Systems.
To run this application a database needs to be running on the local system (using xampp apache and mysql). Running on the default port of 127.0.0.1 or Localhost.
If not on either of these the solution will need to be updated accordingly.
Included is the table for HospManSys with some sample data, you will need to import this into your database as hospmansys.
